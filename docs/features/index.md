# Features

cli_debrid is a full media automation suite. Here's everything it can do.

---

<div class="grid cards" markdown>

- ## :material-view-dashboard:{ .lg .middle } [Dashboard](dashboard.md)
  Statistics, recently aired, upcoming releases, and real-time queue activity.

- ## :material-filmstrip:{ .lg .middle } [Library](library.md)
  Browse and manage your collected media — filter, sort, bulk-delete, and blacklist.

- ## :material-compass:{ .lg .middle } [Discover](discover.md)
  Browse TMDB with advanced filters, FlixPatrol charts, and MDBList collections.

- ## :material-format-list-checks:{ .lg .middle } [Queues](queues.md)
  Real-time view of every item in the processing pipeline.

- ## :material-cloud-download:{ .lg .middle } [Debrid Manager](debrid-manager.md)
  Monitor downloads, manage your debrid library, backup torrents, and run audits.

- ## :material-image-multiple:{ .lg .middle } [Overlays](overlays.md)
  Generate poster overlays with resolution, HDR, audio, and rating badges.

- ## :material-arrow-up-bold:{ .lg .middle } [Upgrade Hub](upgrade-hub.md)
  Find and queue quality upgrades for your existing collection.

- ## :material-robot:{ .lg .middle } [AI Butler](ai-butler.md)
  AI-powered assistant for diagnostics, settings help, and content recommendations.

- ## :material-label:{ .lg .middle } [Plex Labels](plex-labels.md)
  Automatically label Plex items by content source or requester.

- ## :material-chart-line:{ .lg .middle } [Performance](performance.md)
  Real-time CPU and memory monitoring dashboard.

</div>

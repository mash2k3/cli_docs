# cli_debrid

Automated media management powered by debrid services. Automatically scrape, download, and upgrade your media library — movies and TV shows — using Real-Debrid, AllDebrid, Premiumize, Torbox, or Debrid-Link.

<div class="grid cards" markdown>

- ## :material-download-circle:{ .lg .middle } Automated Downloads
  Automatically find and download the best available version of any movie or TV show from your watchlists.

- ## :material-arrow-up-circle:{ .lg .middle } Smart Upgrading
  Continuously monitor and upgrade existing content to better quality when it becomes available.

- ## :material-magnify:{ .lg .middle } Multi-Source Scraping
  Scrape from Zilean, Jackett, Torrentio, Nyaa and more — with configurable quality filters.

- ## :material-server:{ .lg .middle } Multiple Debrid Providers
  Supports Real-Debrid, AllDebrid, Premiumize, Torbox, and Debrid-Link.

- ## :material-television-play:{ .lg .middle } Plex & Jellyfin
  Full integration with Plex and Jellyfin for library management and metadata.

- ## :material-web:{ .lg .middle } Web Interface
  A full-featured web UI for managing your library, settings, and queue — no command line required.
